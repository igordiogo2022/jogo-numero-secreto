# Jogo Número Secreto1️⃣

### 📄Descrição
Jogo advinhe o número secreto seguindo as dicas dadas pelo jogo com o mínimo de tentativas.

<hr>

### 🎮Funcionalidades
- Número sorteado aleatóricamente
- Botão "Chutar
- Botão "Novo jogo"

<hr>

### ❓Como se funciona
Um número aleatório é sorteiado e o objetivo do usuário é advinhar este número com a menor quantidade de tentativas possíveis,
a cada palpite incorreto ele recebe um dica sobre o número secreto.
